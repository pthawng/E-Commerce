# Assets

Thư mục chứa static assets cho back-office:

- `images/` - Hình ảnh (logos, icons, illustrations)
- `fonts/` - Custom fonts
- `videos/` - Video files
- `data/` - Static data files (JSON, CSV, etc.)

## Usage

```typescript
import logo from '@/assets/images/logo.png';
import customFont from '@/assets/fonts/custom-font.woff2';
```
