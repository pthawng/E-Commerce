# Composite Components

Composite components được tạo từ nhiều UI components

## Examples

- `SearchBar` - Kết hợp Input + Button + Dropdown
- `DataTable` - Table với pagination, sorting, filtering
- `FormBuilder` - Dynamic form builder
- `Modal` với custom header/footer

## Usage

```typescript
import { SearchBar, DataTable } from '@/components/composite';
```
